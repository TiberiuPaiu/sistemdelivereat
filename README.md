
Este proyecto es una aplicación web desarrollada para gestionar pedidos de comida a domicilio, desde el registro de clientes y restaurantes hasta la gestión de pedidos y el reparto.

Se ha utilizado el framework Django para su implementación, integrando múltiples funcionalidades para los distintos roles (clientes, repartidores, restaurantes y partners).

En la memoria se detalla el desarrollo completo del proyecto, incluyendo:
- Explicación técnica
- Arquitectura
- Decisiones de diseño
- Demo funcional
